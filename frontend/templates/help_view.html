Help

Dit is onze help pagina hier kan je informatie vinden hoe de app werkt en hoe de api werkt.

Api

Deze api is gemaakt om recepten te leveren uit restaurants uit de regio deventer de api mag je gratis gebruiken klik hier voor de documentatie.

App

De app werkt door een random recept te laten zien uit de regio deventer deze kan varieren van biefstuk tot taco's. Als je gerechten wilt opslaan moet je inloggen als je ingelogd bent kan je de gerechten opslaan door op het +'je te klikken. Als je een nieuw gerecht wilt kan je op de knop nieuw gerecht klikken.
